# Golam Mahadi — Research & Professional Portfolio

A dependency-free, multi-page portfolio designed for GitHub Pages or any static host.

## Pages

- `index.html` — research-first profile and overview
- `research.html` — current research tracks
- `publications.html` — four publications with citation tools
- `projects.html` — filterable project portfolio
- `services.html` — freelance and professional services
- `experience.html` — work, education and skills
- `gallery.html` — filterable non-duplicated image journal
- `contact.html` — offline-friendly project brief form

## Preview locally

From this folder, run:

```bash
python3 -m http.server 8000
```

Open `http://localhost:8000`.

## Deploy

Upload the contents of this folder to the root of the `portfolio2024` GitHub Pages repository. After deployment, submit `sitemap.xml` in Google Search Console and request indexing for the home, research, publications and services pages.

Light mode is the default. Dark mode is optional and saved only after the visitor selects it.
